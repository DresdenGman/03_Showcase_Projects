# Tmall Coupon Usage Prediction

## Project Overview

This project applies machine learning to solve a critical e-commerce business problem: 
predicting which customer segments are most likely to use coupons. The project demonstrates strong business 
orientation and practical application of data science.

Key Highlights:
- Business-focused analysis of coupon usage patterns
- Comprehensive data profiling and cleaning
- Single and multi-variable analysis
- Correlation analysis to identify key factors
- Logistic Regression model development with:
  * Training/test set splitting
  * Model evaluation using accuracy and ROC/AUC metrics
  * Model optimization techniques
- Actionable business recommendations for targeting high-value users

This project showcases the ability to apply data science to real business problems, 
demonstrating both technical competence and business understanding.
