# Airline Customer Value Analysis

## Project Overview

This project demonstrates innovative thinking by developing a custom customer value 
model (LRFMC) that extends the traditional RFM framework. The project showcases deep understanding of 
business requirements and the ability to adapt standard methodologies to specific industry needs.

Key Highlights:
- Innovation: Development of LRFMC model (Length, Recency, Frequency, Mileage, Coefficient) 
  specifically tailored for airline industry
- Business understanding: Recognition that monetary value alone doesn't capture customer value 
  in variable-pricing industries
- Comprehensive data processing pipeline:
  * Data extraction and cleaning
  * Attribute reduction and transformation
  * Feature engineering
- K-means clustering for customer segmentation
- Actionable insights for personalized marketing strategies

This project demonstrates leadership in analytical thinking, creativity in problem-solving, 
and the ability to bridge technical solutions with business needs.
