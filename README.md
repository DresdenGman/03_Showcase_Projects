## 🌟 Projects

### 1. Credit Card Fraud Detection
*Imbalanced Classification · Sampling Strategies · Financial Security*

Handles a highly skewed fraud detection dataset using multiple resampling approaches. Logistic Regression and Decision Tree models are evaluated against precision-recall metrics, with comprehensive comparison of undersampling, oversampling, and SMOTE techniques.

**Techniques:** Logistic Regression · Decision Tree · SMOTE · ROC-AUC · Confusion Matrix

---

### 2. Hotel Booking Analysis & Prediction
*Multi-Dimensional EDA · Ensemble Modeling · Revenue Optimization*

A 10+ dimensional analysis of global hotel booking patterns using Random Forest, XGBoost, and Logistic Regression. Features include seasonal trend analysis, cancellation prediction, and actionable business recommendations for revenue management.

**Techniques:** Random Forest · XGBoost · Logistic Regression · Feature Importance · EDA

---

### 3. Airline Customer Value Analysis
*Innovative LRFMC Model · Customer Segmentation · Business Strategy*

Extends the classic RFM framework into a custom LRFMC model tailored for the airline industry. K-Means clustering segments customers into actionable tiers, with data-driven marketing strategies derived from cluster characteristics.

**Techniques:** K-Means · LRFMC Model · Customer Segmentation · Radar Charts

---

### 4. Bike Sharing Demand Prediction
*Time Series Modeling · Feature Engineering · Algorithm Comparison*

An end-to-end machine learning pipeline comparing SVM and Random Forest for hourly bike rental prediction. Comprehensive feature engineering from datetime, weather, and seasonal data, with cross-validation and feature importance analysis.

**Techniques:** Random Forest · SVM · Cross-Validation · Feature Engineering

---

### 5. Tmall Coupon Usage Prediction
*E-Commerce Analytics · Logistic Regression · Marketing ROI*

Predicts coupon redemption probability using user behavior features from the Tmall platform. Business-oriented model optimization with practical customer targeting recommendations for maximizing marketing ROI.

**Techniques:** Logistic Regression · AUC Optimization · Business Analytics
